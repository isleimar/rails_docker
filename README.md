# rails_docker
Criando um projeto **Ruby on Rails** com Docker

## Pré-requisitos
1. `Docker` instalado.
2. `Docker-Compose` instalado.

## Preparação
Modifique a senha do banco de dados no arquivo ``.env`` 

## Criando o projeto
Execute o script ``create_project``


## Acesso a página
A página estará disponível no endereço:
``http://localhost:3000``
